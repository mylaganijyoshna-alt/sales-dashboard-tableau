# Sales Performance Dashboard (Tableau)

## 📊 Overview
This project is an interactive dashboard created using Tableau to analyze sales performance.

## 🔍 Features
- Region-wise sales analysis
- Product category insights
- Monthly sales trends
- Interactive filters

## 🛠️ Tools Used
- Tableau
- Excel
